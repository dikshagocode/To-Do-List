# ✅ To-Do List Web App
<br>
A simple and functional To-Do List application built using HTML, CSS, and JavaScript. This project allows users to add tasks with deadlines and delete them as needed.

## 🧩 Features
Add tasks with a name and a due date
Delete tasks when completed or no longer needed
Basic input validation
Dynamic task list display

## 🛠️ Technologies Used
HTML – for the page structure
CSS – for styling
JavaScript – for interactivity and dynamic task management

##💡 How to Use
Enter a task name.
Select a date (deadline).
Click the Add button.
The task will appear in the list with its date.
Click the Delete button to remove any task.
